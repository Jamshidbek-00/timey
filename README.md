# Timey Backend

Timey Backend — Django asosida qurilgan backend loyiha.  
Loyiha booking va asosiy biznes logikani API orqali boshqaradi.

## Tech Stack

- Python
- Django
- SQLite (local development)

## Project Structure

- `core/` — asosiy sozlamalar va global config, masters bilan bog'liq
- `bookings/` — booking bilan bog‘liq app
- `config/` — qo‘shimcha konfiguratsiyalar
- `manage.py` — Django boshqaruv fayli
